📄 Portfolio Guide

Everything you need to know to customise, update, and maintain this portfolio. Read this before making any edits.

🌗 Day / Night Mode Toggle

The sun/moon toggle button is in the top-right corner of every page. Click it to switch between Dark Mode (default) and Light Mode. Your preference is saved automatically in your browser so it persists across visits.

✏️ How to Edit Your Name & Info

Open the HTML file in any code editor (VS Code recommended). Search for Akhil Baluni to find your name. Change the tagline in the .hero-tagline paragraph. Edit the About section's text inside the .about-text div.

🚀 How to Add a Project

Find the #projects section. Copy a .project-card block, then replace the placeholder content. The structure you need:

<div class="project-card">
  <div class="project-tag">Category</div>
  <div class="project-title">Project Name</div>
  <div class="project-desc">What it does...</div>
  <div class="project-links">
    <a href="YOUR_LINK" class="project-link">View →</a>
  </div>
</div>

✍️ How to Add a Blog Post

Find the #blogs section. Delete a placeholder and add a real blog card. Use this structure:

<div class="blog-card">
  <div class="blog-card-tag">Category</div>
  <div class="blog-card-title">Post Title Here</div>
  <div class="blog-card-excerpt">Short summary...</div>
  <div class="blog-card-footer">
    <span class="blog-card-date">June 2025</span>
    <a href="YOUR_BLOG_URL" class="blog-card-link">Read →</a>
  </div>
</div>

💡 If you write on Medium, Hashnode, or any platform — just link to the URL of your post in the href.

📚 How to Update Currently Learning

Find the #learning section. Edit or add .learning-card blocks:

<div class="learning-card">
  <div class="learning-icon">🐍</div>
  <div class="learning-title">Topic Name</div>
  <div class="learning-desc">What you're learning...</div>
  <div class="learning-status">Active</div>
</div>

🛠️ How to Add a Skill

In the #skills section, either add a new chip inside an existing .skill-chips div, or add a whole new category card:

<!-- Add a single skill chip -->
<span class="skill-chip">Skill Name</span>

<!-- Add a whole new category -->
<div class="skill-category">
  <div class="skill-category-icon">🔧</div>
  <div class="skill-category-name">Category Name</div>
  <div class="skill-chips">
    <span class="skill-chip">Skill 1</span>
    <span class="skill-chip">Skill 2</span>
  </div>
</div>

🌐 How to Host This Portfolio (Free)

You can host this for free using GitHub Pages:

  1. Create a free account at github.com
  2. Create a new repository named yourusername.github.io
  3. Upload this HTML file, rename it to index.html
  4. Go to Settings → Pages → select the main branch
  5. Your site will be live at yourusername.github.io

Alternative: Netlify.com — drag and drop the HTML file, get a live link instantly. Also completely free!

🎨 How to Change Colors

All colors are defined as CSS variables at the top of the <style> block in :root. To change the accent color (purple #6C63FF), find --accent: and replace the hex code with any color you like.

💡 Pro Tips

  • Use VS Code with the Live Server extension to preview changes in real-time
  • Keep a backup of the HTML file before making big changes
  • Update the Currently Learning section every month to show you're always growing
  • Add real projects as soon as you build them — even small ones count!
  • Share your portfolio link on LinkedIn and in job applications
  • Add a profile photo by replacing the initials badge with an <img> tag
